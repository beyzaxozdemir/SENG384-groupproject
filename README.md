# SENG384-groupproject
# Health AI Platform 

This project is a frontend-based prototype developed for SENG384.

## Features
- User login with .edu email validation
- Role selection (Engineer / Healthcare Professional)
- Post creation, publishing, and deletion
- Dynamic search by city
- Meeting request workflow (conceptual)
- Local storage for data persistence

## Technologies
- HTML
- CSS
- JavaScript

## Notes
- This is a frontend prototype
- No backend or database integration
- Data is stored using browser localStorage

## How to Run
1. Open index.html in a browser
2. Login with a .edu email
3. Start creating and managing posts
